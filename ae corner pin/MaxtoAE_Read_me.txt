Use this script at your own risk
=================================
This script will export the 3d data of a selected object and convert
it into 2D data that can be used in after effect to get an
exact match mover. Very handy for corner pinning in after effects.

Installation
============
-Copy the files into your max script folder.
-Open Max and then go to the utility panel and choose Maxscript
-Choose run script and pick the MaxtoAE script file
